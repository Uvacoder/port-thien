## Overview

## Running Locally

```bash
$ git clone https://github.com/thnmtrn/thien.io.git
$ cd thien.io
$ yarn
$ yarn dev
```

Create a `.env.local` file similar to [`.env.example`](https://github.com/thnmtrn/thien.io/blob/master/.env.example).
